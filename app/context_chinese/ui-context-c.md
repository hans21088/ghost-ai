## 主題

僅深色。無淺色模式。視覺語言是深色技術工作區 - 接近黑色的背景、分層表面和充滿活力的互動元素強調色。

所有顏色定義為 `globals.css` 中的 CSS 自訂屬性，透過 `@theme inline` 映射到 Tailwind 權杖。元件必須使用這些權杖 - 不使用硬編碼的十六進位值或原始 Tailwind 顏色類別（如 `zinc-*`）。


| 角色              | CSS 變數            | 十六進位 / 值               |
|-------------------|------------------------|---------------------------|
| 頁面背景   | `--bg-base`            | `#080809`                 |
| 表面           | `--bg-surface`         | `#111114`                 |
| 提升的表面  | `--bg-elevated`        | `#18181c`                 |
| 微妙表面    | `--bg-subtle`          | `#1e1e23`                 |
| 預設邊框    | `--border-default`     | `#2a2a30`                 | 
| 微妙邊框     | `--border-subtle`      | `#3a3a42`                 |
| 主要文字      | `--text-primary`       | `#f0f0f4`                 |
| 次要文字    | `--text-secondary`     | `#c0c0cc`                 |
| 靜音文字        | `--text-muted`         | `#808090`                 |
| 淡化文字        | `--text-faint`         | `#505060`                 |
| 品牌強調      | `--accent-primary`     | `#00c8d4`（青色）          |
| 品牌淡化         | `--accent-primary-dim` | `rgba(0, 200, 212, 0.12)` |
| AI 強調         | `--accent-ai`          | `#6457f9`（靛藍紫色） |
| AI 文字           | `--accent-ai-text`     | `#8b82ff`                 |
| 錯誤             | `--state-error`        | `#ff4d4f`                 |
| 成功           | `--state-success`     | `#34d3999`                |
| 警告           | `--state-warning`      | `#fbbf24`                 |

Tailwind 工具名稱映射到這些變數。使用 `bg-base`、`bg-surface`、`text-copy-primary`、`text-copy-muted`、`border-surface-border`、`text-brand`、`bg-accent-dim` 等。

## 排版

| 角色      | 字體        | CSS 變數         |
|-----------|------------|----------------------|
| UI 文字   | Geist Sans | `--font-geist-sans`  |
| 代碼/等寬 | Geist Mono | `--font-geist-mono`  |

兩種字體都透過 `next/font/google` 載入，並在 `<html>` 元素上應用為 CSS 變數。基本 `body` 使用具有 `antialiased` 的 Geist Sans。

## 邊框半徑

半徑隨表面深度增加 - 內部元素較小，外部容器較大。

| 環境           | 類別         |
|-------------------|---------------|
| 內聯 / 小型 UI | `rounded-xl`  |
| 卡片 / 面板    | `rounded-2xl` |
| 模態 / 覆蓋層   | `rounded-3xl` |

## 畫布

### 節點顏色調色盤

8 個定義的顏色對。每對指定深色節點填充和充滿活力的對比文字顏色，針對深色畫布上的可讀性進行調整。在 `types/canvas.ts` 中定義為 `NODE_COLORS`

| 節點填充 | 文字顏色 | 字元              |
|-----------|------------|------------------------|
| `#1F1F1F` | `#EDEDED`  | 中立深色（預設） |
| `#10233D` | `#52A8FF`  | 藍色                   |
| `#2E1938` | `#BF7AF0`  | 紫色                 |
| `#331B00` | `#FF990A`  | 橙色                 |
| `#3C1618` | `#FF6166`  | 紅色                    |
| `#3A1726` | `#F75F8F`  | 粉紅色                  |
| `#0F2E18` | `#62C073`  | 綠色                  |
| `#062822` | `#0AC7B4`  | 青藍色                   |

預設節點顏色：`#1F1F1F`，文字為 `#EDEDED`。

### 邊線樣式

平滑步階路徑，帶箭頭標記。預設邊線顏色：`#f8fafc`。筆畫寬度較細 - 邊線在視覺上是節點的次要元素。

## 節點形狀

6 種支持的形狀，在 `types/canvas.ts` 中定義為 `NODE_SHAPES`。複雜形狀（菱形、六邊形、圓柱形）呈現為內聯 SVG，而非 CSS 邊框。

- `rectangle` - 預設通用節點
- `diamond` - 決策 / 閘道
- `circle` - 事件 / 端點
- `pill` - 服務 / 流程
- `cylinder` - 資料庫 / 儲存
- `hexagon` - 外部系統 / 邊界

### 連接句柄

小白色圓形句柄，預設隱藏，在節點懸停時顯示。出現在節點的所有四個邊。

### 畫布背景

React Flow `<Background>` 元件。畫布位於基本背景顏色上。

## 元件程式庫

Tailwind 上的 shadcn/ui。無自訂設計系統。元件位於 `components/ui/`。使用 `shadcn` CLI 添加新元件，而非從頭編寫它們。

## 佈局模式

- 編輯器工作區：全檢視區佈局 - 左側浮動側欄覆蓋層、中心畫布、右側滑出式 AI 側欄。
- 側欄：浮動覆蓋層，具有深色半透明背景和微妙邊框。
- 模態和對話框：居中覆蓋層、`rounded-3xl`、深色背景，帶背景模糊。
- 導覽欄：頂部列表帶深色背景和底部邊框。
